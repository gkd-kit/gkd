GKD 学生入门

面向大学生的上手流程：先导入推荐订阅，再只选择自己已安装、确实需要跳广告的学校常用 App。

适合谁

如果你被运动世界校园、慧生活798、步道乐跑、闪动校园、智慧校园、志愿汇、学习通、易校园等高校常用 App 的启动页、弹窗或流程干扰，这个入口会比完整研究 GKD 规则更省时间。

它不会默认开启全部规则。你需要自己勾选要启用的 App，未选择的 App 会保持关闭。

怎么用

  1. 在首页进入「学生入门」。
  2. 确认应用列表权限和工作模式权限。
  3. 导入推荐订阅。
  4. 在已安装 App 列表里勾选需要跳过广告或弹窗的 App。
  5. 点击应用选择，回到首页后正常使用。

推荐分享方式是发这个说明页和安装包/源码链接，让同学知道它做了什么、需要什么权限、规则来自哪里。不要承诺它能处理所有广告，也不要让别人无脑全开所有 App。
